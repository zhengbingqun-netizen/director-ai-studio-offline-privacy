# Director AI Studio Offline 隐私政策

更新日期：2026-06-29

Director AI Studio Offline 是一个本地运行的短剧大纲、分镜和 AI 视频 Prompt 生成工具。

## 我们收集什么信息

本扩展不收集、上传、出售或共享任何个人信息。

本扩展不会读取用户正在浏览的网页内容，不会收集浏览历史，不会跟踪用户行为，不会访问 Cookie，不会请求账号登录，也不会将用户输入发送到开发者服务器或第三方服务器。

## 用户输入如何被使用

用户在工具中输入的项目标题、角色设定、剧情钩子、视觉风格和角色一致性描述，只会在用户当前浏览器本地用于生成短剧大纲、分镜和 Prompt 文案。

如果用户输入 Pro 授权码或在升级弹窗中填写收码邮箱，本扩展可能会将授权码、待确认的收码邮箱和购买版本保存在用户浏览器的本地存储中，用于保持 Pro 解锁状态或提示用户付款状态。这些信息不会被扩展自动上传到开发者服务器或第三方服务器。

导出的 Markdown、CSV 或 JSON 文件由用户自行保存到本地设备。

## 第三方服务

本扩展不接入第三方 API，不包含远程脚本，不使用分析工具，不使用广告 SDK。

## 可选 Pro 付款

本扩展可能展示可选的 Pro 升级说明和支付宝收款码。付款过程由用户通过支付宝和开发者直接完成，不在扩展内处理支付 API。用户主动发送给开发者的付款截图、邮箱或微信号，仅用于确认付款并发送授权码。

## 数据共享

本扩展不会向任何第三方共享用户数据。

## 数据保留

本扩展不建立开发者服务器，也不保存用户数据到远程位置。用户导出的文件由用户自行管理和删除。
